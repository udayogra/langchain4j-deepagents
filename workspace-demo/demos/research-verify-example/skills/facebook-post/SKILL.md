# Facebook post skill

Transform **`research/brief.md`** into **`drafts/facebook.md`**.

## Voice
- Warm, **community-oriented** — like explaining to friends or a group.
- Shorter paragraphs (1–3 sentences); extra line breaks help feed readability.
- Optional **one** question at the end to invite comments.

## Length
- **~400–900 characters** is a good band for a Page-style update; can go a bit longer if the brief is rich, but stay skimmable.

## Hashtags and links
- **0–3 hashtags** only if they fit naturally; Facebook audiences often prefer none.
- Do not paste fake links.

## Safety
- Honest about limitations (general knowledge vs sourced).

## Output file
- **`drafts/facebook.md`**
