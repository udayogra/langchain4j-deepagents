# LinkedIn post skill

Transform **`research/brief.md`** (after verification passed) into **`drafts/linkedin.md`**.

## Voice and shape
- Professional, first person or neutral “we” — avoid hype adjectives.
- **Opening line** = clear hook (why this matters to the reader).
- **Body** = 2–4 short paragraphs OR tight bullets for scannability on mobile.
- **Close** = one question or soft CTA (comment / follow-up) — no engagement bait (“like if you agree”).
- **Hashtags** = 3–5 max at the end, relevant only; no hashtag stuffing.

## Length
- Aim **~900–1,300 characters** for the main text (excluding hashtags line). Trim if longer.

## Safety
- Do not claim live data, surveys, or URLs you did not fetch. If the brief says “general knowledge,” say that lightly in copy only if needed for honesty.

## Output file
- Write **`drafts/linkedin.md`** as the final post text (Markdown ok: use `##` sparingly or plain paragraphs).
