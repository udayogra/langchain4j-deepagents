# Twitter / X post skill

Transform **`research/brief.md`** into **`drafts/twitter.md`**.

## Format (single post)
- **One** core insight or takeaway; conversational but crisp.
- Target **under 260 characters** for the main tweet body so there is room if the user adds a link later.
- **0–2 hashtags** inline or at the end; prefer none if the line is already dense.
- No threads in this demo unless the brief explicitly demands two tweets — then use `Tweet 1:` / `Tweet 2:` lines in the file.

## Tone
- Direct, human, slightly informal vs LinkedIn — still accurate. No ALL CAPS emphasis spam.

## Safety
- Same as LinkedIn: no fake stats or invented URLs.

## Output file
- **`drafts/twitter.md`** — plain text or minimal Markdown.
